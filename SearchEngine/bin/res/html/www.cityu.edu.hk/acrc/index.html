一、研究领域

建立在香港城市大学深圳研究院的建筑与土木工程研发中心拥有多个学科领域，包括土木与结构工程，屋宇设备工程，工程管理，城市规划，建筑学，屋宇测量等领域的专家学者，致力于发挥多学科交叉的专业特色和科研优势，为满足多样化的社会及工程界的需求提供以下高质量的科研成果和咨询服务。

1. 土木工程

  • 结构工程
  • 建筑材料
  • 计算力学
  • 结构非线性分析
  • 结构健康监测和优化
  • 纳米力学和智能材料

2. 屋宇设备与建筑环境

  • 可持续发展建筑
  • 室内空气质量评估和污染物分析
  • 建筑节能
  • 建筑通风
  • 振动噪声
  • 建筑采光
  • 火灾动力模拟
  • 复杂性人流模拟
  • 建筑人员疏散

3. 建筑风致效应评估

  • 现场实测
  • 风洞测试
  • 数值模拟

4. 工程管理和工程测量

  • 生产力研究
  • 争议解决和合同管理
  • 安全管理
  • 绩效评估方法
  • 楼宇维护和维护成本评估
  • 房屋定价策略
  • 建筑经济学
  • 古建筑维护方案

二、研发中心的工作范围和发展目标

  1. 成为国内和国际公认一流的建筑与土木工程科研中心；
  2. 促进产学研转化的科研孵化器；
  3. 研究生和博士后的培训基地；
  4. 促进和内地高校的博士生联合培养项目；
  5. 争取内地政府，研究机构,工业和企业界在建筑与土木工程研究方面的经费和资金支持；
  6. 加强与内地高校，科研院所和工业企业的合作；
  7. 为社会及工程界提供高质量的科研成果和咨询服务。
